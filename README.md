# 嵌入式系统环境数据采集项目
本项目旨在设计一个基础嵌入式系统，实现环境数据采集，用于学习与实践。主要功能为采集环境中的温度数据，并进行简单展示。

### 作业需求
1. 利用合适的硬件设备及接口，实现环境温度数据的采集。
2. 通过软件编程，将采集到的数据进行处理和输出展示。
